Reports a package that is exported or opened to the same Java 9 module where it's defined. The quick-fix removes such directives from module-info.java

Example:

  module com.mycomp {
    exports com.mycomp.main to com.mycomp;
  }

After the quick-fix is applied the result looks like:

  module main {
  }

This inspection only reports if the language level of the project or module is 9 or higher.
