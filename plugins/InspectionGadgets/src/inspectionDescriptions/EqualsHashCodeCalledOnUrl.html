Reports equals() or hashCode() being called on java.net.URL objects. This can cause performance problems because those methods uses a DNS lookup to determine the equality of two java.net.URL objects. Depending on the availability of the network and the speed of the DNS server, this can cause significant delays.

Example:

  int f(URL url1, URL url2) {
    if (url1.equals(url2)) return url1.hashCode();
    else return url2.hashCode();
  }
