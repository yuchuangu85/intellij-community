Reports while loops which iterate over collections, and can be replaced with more concise enhanced for loop (i.e. foreach iteration syntax).

Example:

  Iterator it = c.iterator();
  while(it.hasNext()) {
    Object obj = it.next();
    System.out.println(obj);
  }

Can be replaced with:

  for (Object obj : c) {
    System.out.println(obj);
  }

This inspection only reports if the language level of the project or module is 5 or higher
