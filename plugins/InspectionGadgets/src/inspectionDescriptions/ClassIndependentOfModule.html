Reports any class in module which:
  • doesn't depend on any other class in this module
  • is not a dependency for any other class in this module
Such classes are an indication of ad-hoc or incoherent modularisation strategies, and may often profitably be moved.

Because this inspection requires global code analysis it is only available for Analyze|Inspect Code or Analyze|Run Inspection by Name and it will not report in the editor.
