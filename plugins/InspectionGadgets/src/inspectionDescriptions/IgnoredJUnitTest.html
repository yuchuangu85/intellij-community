Reports usages of JUnit 4's @Ignore or JUnit 5's @Disabled annotations. Test annotated with these annotations for a long time, especially without a specified reason, are a code smell.

For example:


  @Ignore
  public class UrgentTest {

    @Test
    public void testIt() {
      Assert.assertEquals("expected", "actual");
    }
  }

Use the check box below to only report when no reason is specified as the annotation's value attribute.
