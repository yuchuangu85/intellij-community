Reports lambdas that can be replaced with method references. While it could be a matter of taste often, method references are more clear and readable, compared to lambdas.

Example:

  Runnable r = () -> System.out.println();

After the quick-fix is applied the result looks like:

  Runnable r = System.out::println;

The inspection may suggest method references even if lambda doesn't call any method, like replacing obj -> obj != null with Objects::nonNull. Use the Settings | Editor | Code Style | Java | Code Generation settings to configure special method references.

This inspection only reports if the language level of the project or module is 8 or higher
