Reports on field accesses which are not qualified with this or some other qualifier. Some coding styles mandate that all field accesses are qualified to prevent confusion with local variable or parameter accesses.

Example:

  class Foo {
    int foo;

    void bar() {
      foo += 1;
    }
  }

After the quick-fix is applied, the result looks like this:

  class Foo {
    int foo;

    void bar() {
      this.foo += 1;
    }
  }
