Reports usages of @TestOnly and @VisibleForTesting annotated methods and classes in production code. Also reports usage of applying @TestOnly @VisibleForTesting to the same element.
The problems are not reported if such method or class is referenced from:
  • Code under 'Test Sources' folder
  • A test-class (JUnit/TestNG)
  • Another @TestOnly-annotated method

Example (in production code):

  @TestOnly
  fun foo() { ... }

  fun main () {
    foo()
  }
