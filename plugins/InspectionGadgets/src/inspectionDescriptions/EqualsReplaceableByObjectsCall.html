Reports expressions that can be replaced with a call to java.util.Objects#equals.

Example:

  void f(Object a, Object b) {
    boolean result = a != null && a.equals(b);
  }

After the quick-fix is applied the result looks like:

  void f(Object a, Object b) {
    boolean result = Objects.equals(a, b);
  }

Replacing expressions like a != null && a.equals(b) with Objects.equals(a, b) slightly changes the semantics, but if that's what you need there's a check box for that in the inspection settings.

This inspection only reports if the language level of the project or module is 7 or higher
