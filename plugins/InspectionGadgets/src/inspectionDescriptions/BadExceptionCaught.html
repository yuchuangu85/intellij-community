Reports catch clauses which catch inappropriate exceptions. Some exceptions, for instance java.lang.NullPointerException and java.lang.IllegalMonitorStateException represent programming errors and so should almost certainly not be caught in production code.

Example:

  try {
    return component.getMousePosition(true) != null;
  }
  catch (NullPointerException e) { // warning: Prohibited exception 'NullPointerException' caught
    return false;
  }
