Reports calls to one of the JUnit assertEquals() methods with arguments of type array. This will compare the arrays' identity instead of the array contents. Array contents should be checked with one of the assertArrayEquals() methods.

For example:


  @Test
  public void testSort() {
    int[] actual = {248, 496, 0, 56};
    Arrays.sort(actual);
    Assert.assertEquals(new int[] {0, 56, 248, 496}, actual);
  }

A quick fix is provided to perform the replacement:


  @Test
  public void testSort() {
    int[] actual = {248, 496, 0, 56};
    Arrays.sort(actual);
    Assert.assertArrayEquals(new int[] {0, 56, 248, 496}, actual);
  }
