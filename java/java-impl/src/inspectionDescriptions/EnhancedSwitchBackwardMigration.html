Reports enhanced switch statements and expressions, and suggests replacing them with a traditional switch statement.

Example:

  boolean even = switch (condition) {
    case 1, 3, 5, 7, 9 -> false;
    default -> true;
  };

After the quick-fix is applied the result looks like:

  boolean even;
  switch (condition) {
    case 1:
    case 3:
    case 5:
    case 7:
    case 9:
      even = false;
      break;
    default:
      even = true;
      break;
}
