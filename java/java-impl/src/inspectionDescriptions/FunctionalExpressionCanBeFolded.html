Reports method references or lambda expressions which point to a method of their own functional interface type and hence can be replaced with their qualifiers removing unnecessary object allocation.

For example:

  SwingUtilities.invokeLater(r::run);
  SwingUtilities.invokeAndWait(() -> r.run());

After the quick-fix is applied the result looks like:

  SwingUtilities.invokeLater(r);
  SwingUtilities.invokeAndWait(r);

Reports only if the language level of the project or module is 8 or higher
