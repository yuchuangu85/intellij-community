Reports pointless or pointlessly complicated boolean expressions. Such expressions include &&-ing with true, ||-ing with false, equality comparison with a boolean literal, or negation of a boolean literal. Such expressions can be simplified.

Example: 


  boolean a = !(x && false);
  boolean b = false || x;
  boolean c = x != true;

After the quick fix is applied the result looks like: 


  boolean a = true;
  boolean b = x;
  boolean c = !x;

Use the checkbox below to ignore named constants when determining if an expression is pointless.
