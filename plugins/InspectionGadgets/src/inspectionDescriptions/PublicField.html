Reports public fields. Constants (i.e. fields that are marked static and final) are not reported.

Example:

  class Main {
    public String name;
  }

After the quick-fix is applied the result looks like:

  class Main {
    private String name;

    public String getName() {
        return name;
    }

    public void setName(String name) {
        this.name = name;
    }
  }
