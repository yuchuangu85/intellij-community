Reports size == 0 checks of Collections/Array/String that should be replaced with isEmpty().

Using a corresponding functions makes your code simpler.

The quick-fix replaces a size zero check with isEmpty().

Example:


  fun foo() {
      val arrayOf = arrayOf(1, 2, 3)
      arrayOf.size == 0
  }

After the quick-fix is applied:


  fun foo() {
      val arrayOf = arrayOf(1, 2, 3)
      arrayOf.isEmpty()
  }
