Reports functional interfaces methods calls which are directly invoked on the definition of the lambda, method reference or anonymous class. Such method calls can be replaced with the body of the functional interface implementation.

Example:

  boolean contains(List<String> names, String name) {
    return ((Predicate<String>)x -> {
      return names.contains(x);
    }).test(name);
  }

After the quick-fix is applied the result looks like:

  boolean contains(List<String> names, String name) {
    return names.contains(name);
  }
