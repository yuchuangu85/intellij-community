Reports assert messages which are not of type java.lang.String. By using a string, more information can be provided to help diagnose the failure or the reason for the assertion.

Example:

  void foo(List<Integer> myList) {
    assert myList.isEmpty() : false;
  }

Use the checkbox below to only warn when the assert message type is boolean or java.lang.Boolean. A boolean detail message is unlikely to provide additional information about an assertion failure, and could possibly be the result of a mistakenly entered : instead of &.
