Reports calls to equals() or compareTo() were an object is compared for equality with itself. This means the argument and the qualifier to the call are identical, and it will always return true for equals() or always 0 for compareTo(). Also reports call to static methods Objects.equals(), Objects.deepEquals(), Arrays.equals(), Comparator.compare and similar methods with two identical arguments.

Example:

  class Foo {
    boolean foo(Object o) {
        return o.equals(o); // warning
    }

    boolean bar(String[] ss) {
        return Arrays.equals(ss, ss); // warning
    }
}
