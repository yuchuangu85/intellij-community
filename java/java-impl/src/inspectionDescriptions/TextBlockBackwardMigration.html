Suggests to replace text blocks with regular string literals. This could be part of an effort to migrate the code back to an earlier version of Java than 15.

Example: 


  Object obj = engine.eval("""
                           function hello() {
                             print('"Hello, world"');
                           }

                           hello();
                           """);

After the quick fix is applied the result looks like: 


  Object obj = engine.eval("function hello() {\n" +
                           "  print('\"Hello, world\"');\n" +
                           "}\n" +
                           "\n" +
                           "hello();\n");

This inspection only reports if the configured language level is 15 or higher.

New in 2019.3
