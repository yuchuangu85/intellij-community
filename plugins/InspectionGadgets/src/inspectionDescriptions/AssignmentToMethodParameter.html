Reports assignment to a variable declared as a method parameter. It also reports any attempt to increment or decrement the variable. While occasionally intended, this construct can be confusing, and disallowed in some Java projects.

Example:

  void printTrimmed(String s) {
    s = s.trim();
    System.out.println(s);
  }

A quick-fix is suggested to declare a new variable:

  void printTrimmed(String s) {
    String trimmed = s.trim();
    System.out.println(trimmed);
  }
