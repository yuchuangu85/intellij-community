Reports any package.html files which are used for documenting packages. Since JDK 1.5 it is recommended to use package-info.java files instead, since such files can also contain package annotations. In this way, package-info.java becomes the sole repository for package level annotations and documentation.

Example: package.html

<html>
  <body>
    Documentation example.
  </body>
</html>

After the quick-fix is applied, the result looks like this: package-info.java

/**
 * Documentation example.
 */
package com.sample;
