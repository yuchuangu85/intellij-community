Reports assignments to a lambda parameter, this includes compound assignments and incrementing or decrementing the parameter. While occasionally intended, this construct can be confusing, and is often the result of a typo, or wrong variable being used.

Example:

  list.forEach(s -> {
    s = s.trim();
    System.out.println("String: " + s);
  });

A quick-fix is suggested to declare a new variable:

  list.forEach(s -> {
    String trimmed = s.trim();
    System.out.println("String: " + trimmed);
  });
