Reports all calls to parametrized methods where explicit argument types can be omitted, since they will be unambiguously inferred by compiler.

Use of redundant type arguments makes code excessive and thus less readable.

Example:

  List<String> list = Arrays.<String>asList("Hello", "World");

A quick-fix is provided to remove redundant type arguments:

  List<String> list = Arrays.asList("Hello", "World");
