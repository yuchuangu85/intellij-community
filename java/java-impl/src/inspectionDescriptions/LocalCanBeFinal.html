Reports parameters or local variables, found in the specified inspection scope, that may have a final modifier added.

Example:

  ArrayList<Integer> list = new ArrayList();
  fill(list);
  return list;

After the quick-fix is applied, the result looks like this:

  final ArrayList<Integer> list = new ArrayList();
  fill(list);
  return list;

Use the checkboxes below to define whether parameters or local variables should be reported.
