Reports method calls whose target is another method call.

Example:

  class X { Y foo() { return new Y(); } }
  class Y { X bar() { return new X(); } }

  void foo() {
    Y y = new X().foo(); // Chained method call
  }

Use the first checkbox below to allow chained method calls in field initializers, for instance private Y y = new X().foo();.

Use the second checkbox below to allow chained method calls operating on a unique type, for instance new StringBuilder().append("x: ").append(new X()).append("y: ").append(new Y()).toString();.
