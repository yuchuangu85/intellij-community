Reports try-finally statements which can use Java 7 Automatic Resource Management that is less error-prone. A quick-fix is available to convert the try-finally statement into a try-with-resources statement.

Example:

  PrintStream printStream = new PrintStream(fileName);
  try {
    printStream.print(true);
  } finally {
    printStream.close();
  }

A quick-fix is provided to pass the cause to constructor:

  try (PrintStream printStream = new PrintStream(fileName)) {
    printStream.print(true);
  }

This inspection only reports if the language level of the project or module is 7 or higher
