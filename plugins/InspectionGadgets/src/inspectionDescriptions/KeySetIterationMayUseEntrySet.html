Reports iteration over the keySet() of a java.util.Map instance, where the iterated keys are used to retrieve the values from the map. Such iteration may be more efficient when replaced by iteration over the entrySet() or values() (if key is not actually used). Similarly keySet().forEach(key -> ...) can be replaced with forEach((key, value) -> ...) if values are retrieved inside lambda.

Example:

for (Object key : map.keySet()) {
   Object val = map.get(key);
}

After the quick-fix is applied:

for (Object val : map.values()) {}
