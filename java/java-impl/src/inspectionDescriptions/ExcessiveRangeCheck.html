Reports condition chains where a value range is checked and where these condition chains can be simplified to a single check.

The quick-fix replaces the condition chain with the simplified expression:

Before:

  x > 2 && x < 4

After:

  x == 3

Before:

  arr.length == 0 || arr.length > 1

After:

  arr.length != 1

New in 2019.1
