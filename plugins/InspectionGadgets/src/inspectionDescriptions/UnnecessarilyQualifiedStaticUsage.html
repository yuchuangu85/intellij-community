Reports calls to static methods or accesses of static fields on the current class which are qualified with the class name. Such qualification is unnecessary, and may be safely removed.

Example:

  class Foo {
    static void foo() {}
    static int x;

    void bar() {
      Foo.foo();
      System.out.println(Foo.x);
    }

    static void baz() { Foo.foo(); }
  }

After the quick-fix is applied, the result looks like this:

  class Foo {
    static void foo() {}
    static int x;

    void bar() {
      foo();
      System.out.println(x);
    }

    static void baz() { foo(); }
  }

Use the first checkbox below to ignore accesses of static fields. In the above example, void bar() { System.out.println(Foo.x); } will not be reported.

Use the second checkbox below to ignore calls to static methods. In the above example, void bar() { Foo.foo(); } will not be reported.

Use the third checkbox below to only report calls to static methods or accesses of static fields from a static context. In the above example, only static void baz() { Foo.foo(); } will be reported.
