Reports on allocations of arrays with known lengths of zero when there is a constant for that in the class of array's element type. As zero-length arrays as immutable, you may save memory reusing the same array instance.

Example:

  class Item {
    // Public zero-length array constant that can be reused 
    public static final Item[] EMPTY_ARRAY = new Item[0];
  }
  class EmptyNode {
    Item[] getChildren() {
      // Unnecessary zero-length array creation
      return new Item[0];
    }
  }

After the quick-fix is applied the result looks like:

  class EmptyNode {
    Item[] getChildren() {
      return Item.EMPTY_ARRAY;
    }
  }
