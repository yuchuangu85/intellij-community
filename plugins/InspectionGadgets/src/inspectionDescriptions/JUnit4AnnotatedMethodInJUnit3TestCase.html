Reports JUnit 4 @Test annotated methods inside inheritor of junit.framework.TestCase. Mixing JUnit API's like this is confusing and can lead to problems running the tests.

Example:

  public class MyTest extends TestCase {
      @Test //name doesn't start from "test", thus would be ignored
      public void wouldBeIgnored() {}
  
      @Test   //name starts from "test"
      @Ignore //thus would be executed despite @Ignore annotation
      public void testWouldBeExecuted() {}
  }

Provided fixes:

  • Remove @Ignore annotation and rename the test method, so the name doesn't start with "test"
  • Convert JUnit 3 test class to JUnit 4
