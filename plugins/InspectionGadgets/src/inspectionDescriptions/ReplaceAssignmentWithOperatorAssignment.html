Reports assignment operations which can be replaced by operator-assignment. Code using operator assignment is shorter and may be clearer.

Example: 

  x = x + 3;
  x = x / 3;

After the quick fix is applied the result looks like: 

  x += 3;
  x /= 3;

Use the checkbox below to ignore the conditional operators && and ||. Replacing conditional operators with operator assignment modifies the semantics of the expression.
