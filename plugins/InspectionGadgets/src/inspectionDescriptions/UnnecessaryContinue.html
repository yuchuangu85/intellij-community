Reports continue statements if they are last reachable statements in the loop. These continue statements are unnecessary and can be safely removed.

Example:

  for (String element: elements) {
    System.out.println();
    continue;
  }

After the quick-fix is applied the result looks like:

  for (String element: elements) {
    System.out.println();
  }

The inspection doesn't analyze the JSP files.

Use the checkbox below to ignore continue statements when they are placed in 'then' branch of a complete if-else statement.

Example:

  for (String element: elements) {
    if(element.isEmpty()) {
      continue;
    } else {
      //...
    }
  }
