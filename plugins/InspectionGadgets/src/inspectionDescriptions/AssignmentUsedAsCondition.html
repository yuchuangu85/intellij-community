Reports an assignment being used as the condition of an if, while, for or do statement, or in any other context. While occasionally intended, this usage is confusing, and may indicate a typo (= used instead of ==).

Example:


  void update(String str, boolean empty) {
    // Warning: 'empty' is reassigned,
    // not compared to str.isEmpty()
    if (empty = str.isEmpty()) {
      ...
    }
  }

A quick-fix is suggested to replace = with ==:


  void update(String str, boolean empty) {
    if (empty == str.isEmpty()) {
      ...
    }
  }
