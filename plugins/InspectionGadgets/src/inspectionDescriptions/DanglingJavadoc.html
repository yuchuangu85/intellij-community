Reports Javadoc comments if they don't belong to any class, method or field. Dangling Javadoc comments are ignored by the Javadoc tool and will not become part of the html documentation it generates.

Example:

  class A {
    /**
     * Dangling comment
     */
     /**
      * Method javadoc
      */
      public void m(){}
  }

Quickfix may delete such comment completely or convert to a block comment, like: 

  class A {
  /*
    Dangling comment
   */
  /**
   * Method javadoc
   */
  public void m(){}
}
