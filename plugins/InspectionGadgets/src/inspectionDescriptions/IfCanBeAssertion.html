Reports if statements which only throw a java.lang.Throwable from the then branch, and don't have an else branch. Such statements can be converted to more compact assert statements. Also reports Guava's Preconditions.checkNotNull(). These can be replaced with a Objects.requireNonNull() call, which may avoid the need for a library.

Example: 


  if (x == 2) throw new RuntimeException("fail");
  if (y == null) throw new AssertionError();
  Preconditions.checkNotNull(z, "z");

After the quick fix is applied the result looks like:


  assert x != 2 : "fail";
  Objects.requireNonNull(y);
  Objects.requireNonNull(z, "z");

By default this inspection does not highlight in the editor, but only provides a quick fix.
