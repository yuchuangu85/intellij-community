Suggests replacing null check with Objects/Stream static method.

Example:

if (message == null) {
  application.messageStorage().save(new EmptyMessage());
} else {
  application.messageStorage().save(message);
}

After the quick-fix is applied the result looks like:

  application.messageStorage()
    .save(Objects.requireNonNullElseGet(message, () -> new EmptyMessage()));

Use the checkbox below to not warn if length of replacement bigger than original.
