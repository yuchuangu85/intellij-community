Reports TestNG's org.testng.annotations.Configuration annotation.

It's better to configure a test suite with the modern @BeforeXXX/@AfterXXX annotations.

Example:

  public class Test {
    @Configuration(beforeSuite = true, afterTest = true)
    public void afterBefore(){
    }
  }

After the quick-fix is applied:

  public class Test {
    @AfterTest()
    @BeforeSuite()
    public void afterBefore(){
    }
  }
