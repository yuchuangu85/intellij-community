Reports strings in method parameters and return values annotated with @Nls annotation with "capitalization" parameter to confirm with capitalization rules, which is required by most platform UI guidelines.

Example:


  void setTitle(@NlsContexts.DialogTitle String title) {}
  setTitle("This is sentence capitalization but should be title");

After the quick-fix is applied, the result looks like this:


  setTitle("This Is Sentence Capitalization but Should Be Title");
