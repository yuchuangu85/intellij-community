Reports any unnecessary unary minuses. Such expressions could be hard to understand and bug-prone.

Example 1:

  void unaryMinus(int i) {
    int x = - -i;
  }

Two quick fixes are suggested here:

1) Remove - operators before i variable: 
  void unaryMinus(int i) {
    int x = i;
  }
2) Replace - operators with prefix decrement operator: 
  void unaryMinus(int i) {
    int x = --i;
  }

Example 2:

  void unaryMinus(int i) {
    i += - 8;
  }

After the quick-fix is applied, the result looks like this:

  void unaryMinus(int i) {
    i -= 8;
  }
