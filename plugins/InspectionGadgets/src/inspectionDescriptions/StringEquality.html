Reports any use of == or != to compare strings, which compares object identity. In most cases strings should be compared with an equals() call instead, which does a character-by-character comparison when the strings are different objects.

Example:

  void foo(String s, String t) {
    final boolean b = t == s;
  }

If t is known to be non null, then it's safe to apply "unsafe" quickfix and get the result like:

  void foo(String s, String t) {
    final boolean b = t.equals(s);
  }
