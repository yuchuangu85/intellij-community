Reports any instanceof expressions on catch block parameters. Testing the type of catch parameters is usually better done by having separate catch blocks, rather than instanceof.

Example:

  void foo(Runnable runnable) {
    try {
        runnable.run();
    } catch (Throwable throwable) {
        if (throwable instanceof NoClassDefFoundError) { // warning: 'instanceof' on 'catch' parameter 'throwable'
            System.out.println("Class not found!");
        }
    }
  }
