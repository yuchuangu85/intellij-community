Reports on any call to wait() not made inside a loop. wait() is normally used to suspend a thread until a condition is true. Because the thread could have been waken up for a different reason, that condition should be checked after the wait() returns. A loop is a simple way to achieve this.

Example:


  class BoundedCounter {
    private int count;
    synchronized void inc() throws InterruptedException {
      // condition not checked after wake-up
      if (count >= 10) wait();
      ++count;
    }
  }
