Reports any references to inner classes which are unnecessarily qualified with the name of the enclosing class. Such qualification is unnecessary, and may be safely removed. This may require the addition of an import for the inner class.

Example:

  class X {
     X.Y foo;
     class Y{}
  }

After the quick-fix is applied the result looks like:

  class X {
     Y foo;
     class Y{}
  }

Use the checkbox below to ignore references to inner classes where the removal of the qualification would require the addition of an import.
