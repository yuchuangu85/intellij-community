Reports copy constructors that don't copy all fields in the class. Fields with the transient modifier are considered unnecessary to copy. Same goes for final fields with initializers.

Example:

  class Point {

    private int x;
    private int y;

    Point(int x, int y) {
      this.x = x;
      this.y = y;
    }

    Point(Point other) {
    }
  }

In this example inspection will report that fields x and y are not initialized in Point copy constructor.
