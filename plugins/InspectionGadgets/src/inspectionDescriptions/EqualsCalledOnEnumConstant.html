Reports equals() calls on enum constants. Such calls can be replaced by an identity comparison (==) because two enum constants are equal only when they have the same identity. Use the quick-fix to change the call to a comparison.

Example:

  boolean foo(MyEnum value) {
    return value.equals(MyEnum.FOO);
  }

After the quick-fix is applied, the result looks like this:

  boolean foo(MyEnum value) {
    return value == MyEnum.FOO;
  }
