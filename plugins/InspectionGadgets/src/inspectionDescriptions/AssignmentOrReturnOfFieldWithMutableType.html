Reports any attempt to return or assign an array, Collection, Date, Map, Calendar, etc. field from a method parameter. Since such types are mutable, this construct may result in an object having its state modified unexpectedly from outside the owning class. While occasionally useful for performance reasons, this construct is inherently bug prone.

The inspection reports usage of following mutable types:

  • java.util.Date
  • java.util.Calendar
  • java.util.Collection
  • java.util.Map
  • com.google.common.collect.Multimap
  • com.google.common.collect.Table

Use the checkbox below to ignore assignments and returns in private methods.
