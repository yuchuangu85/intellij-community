Reports annotations with SOURCE or CLASS retention policy when they are supposed to be used by JUnit 5. Note, that if retention policy is not specified, then the default retention policy CLASS is used. Such annotations are not available at runtime and most probably their retention policy should be fixed to be accessible via reflection.

Example:

  @Testable
  public @interface UnitTest {}

After the quickfix is applied the result looks like:

  @Retention(RetentionPolicy.RUNTIME)
  @Testable
  public @interface UnitTest {}

New in 2017.2
