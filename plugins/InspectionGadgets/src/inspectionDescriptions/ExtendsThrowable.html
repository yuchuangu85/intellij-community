Reports any classes which directly extend java.lang.Throwable. It is generally considered bad practice to extend java.lang.Throwable directly and it is usually enough to extend either java.lang.RuntimeException or java.lang.Exception or in special cases java.lang.Error.

Example:

  class EnigmaThrowable extends Throwable {} // warning: Class 'EnigmaThrowable' directly extends 'java.lang.Throwable'
