Reports non-static inner classes that implement Serializable and are inside of a class that doesn't implement Serializable.

Such classes are unlikely to serialize correctly due to implicit references to the outer class.

Example:

  class A {
    class Main implements Serializable {
    }
  }

Use the Options panel below to configure the inspection:

  • Fill the table below to suppress the reporting subclasses of a class that implement Serializable as they are not intended to be serialized.
  • Enable the checkbox below to ignore Serializable anonymous classes.
