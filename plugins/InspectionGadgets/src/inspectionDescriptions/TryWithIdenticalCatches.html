Reports identical catch sections in a single try statement. Such sections can be collapsed into a multi-catch section. This reduces code duplication and prevents situations when one catch section is updated though another one is not.

Example:

    try {
        doSmth();
    }
    catch (IOException e) {
        LOG.error(e);
    }
    catch (URISyntaxException e) {
        LOG.error(e);
    }

The quickfix changes this code to the compact form:

    try {
        doSmth();
    }
    catch (IOException | URISyntaxException e) {
        LOG.error(e);
    }

This inspection only reports if the language level of the project or module is 7 or higher
