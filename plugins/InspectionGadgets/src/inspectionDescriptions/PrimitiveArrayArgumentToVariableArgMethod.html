Reports any calls to a variable arity method where the call has a primitive array in the variable arity parameter position (e.g System.out.printf("%s", new int[]{1, 2, 3}) ). Such a primitive-array argument may be confusing, as it will be wrapped as a single-element array, rather than each individual element being boxed, as might be expected.

Example: 

  String.format("%s", new int[]{1, 2, 3});

After applying the fix: 

  String.format("%s", (Object) new int[]{1, 2, 3});
