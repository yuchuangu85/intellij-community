Reports if statements which can be simplified to a single assignment, return or assert statement.

For example: 


    if (foo()) {
       return true;
    } else {
       return false;
    }
can be simplified to 

    return foo();

Use the option below to not display a warning in case of chained if statement. For example: 


    if (condition1) return true;
    if (condition2) return false;
    return true;

The fix action will still be available in this case.
