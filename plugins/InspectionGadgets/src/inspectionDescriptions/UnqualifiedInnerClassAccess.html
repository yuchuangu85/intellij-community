Reports any references to inner classes which are not qualified with the name of the enclosing class.

Example:

  import foo.Foo.Bar;

  class Foo {
    class Bar {}
  }

  class Baz {
    void f(Bar bar) {}
  }

After the quick-fix is applied, the result looks like this:

  class Foo {
    class Bar {}
  }

  class Baz {
    void f(Foo.Bar bar) {}
  }

Use the checkbox below to ignore references to inner classes contained within the same class, which therefore do not require an import.
