Reports switch statements/expressions where the default case comes before some other case. This construct is unnecessarily confusing. There is a fix that moves default case to the last position. The fix is only available when given branch has break/yield at the end.

Example:

  switch (n) {
      default:
          System.out.println();
          break;
      case 1:
          break;
  }

After applying the fix:

  switch (n) {
    case 1:
        break;
    default:
        System.out.println();
        break;
  }
