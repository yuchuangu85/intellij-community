Reports logger fields on classes which are not declared static and final. Ensuring that every classes logger is effectively constant and bound to that class simplifies the task of providing a unified logging implementation for an application. Interfaces, enumerations, annotations and inner classes are not reported by this inspection.

For example:


  public class Significant {
    private Logger LOG = Logger.getLogger(Critical.class);
  }

A quick fix is provided to change the logger modifiers to static final:


  public class Significant {
    private static final Logger LOG = Logger.getLogger(Critical.class);
  }
