Reports the enhanced for statements. Such for statements are not supported under Java 1.4 and older.

Example:

  for (int x: Arrays.asList(1, 2, 3)) {
    System.out.println(x);
  }

After the quick-fix is applied, the result looks like this: 

  for (Iterator<Integer> iterator = Arrays.asList(1, 2, 3).iterator(); iterator.hasNext(); ) {
    final int x = iterator.next();
    System.out.println(x);
  }
