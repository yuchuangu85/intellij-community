Reports calls to one of JUnit's assertEquals() methods which can be replaced with an equivalent call to assertSame(). This is possible when the arguments are instances of a final class which does not override the Object.equals() method and makes it explicit that object identity is compared.

For example:


  @Test
  public void testObjectType() {
    Object o = getObject();
    Assert.assertEquals(String.class, o.getClass());
  }

A quick fix is provided to perform the replacement:


  @Test
  public void testSort() {
    Object o = getObject();
    Assert.assertSame(String.class, o.getClass());
  }
