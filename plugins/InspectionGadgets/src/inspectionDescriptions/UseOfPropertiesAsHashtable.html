Reports calls to the following methods:
  • java.util.Hashtable#put
  • java.util.Hashtable#putIfAbsent
  • java.util.Hashtable#putAll
  • java.util.Hashtable#get
on a java.util.Properties object.

For reasons lost to history, java.util.Properties inherits java.util.Hashtable, but using those methods is discouraged to prevent corruption of properties values of types other than String.

Example:

  Object f(Properties props) {
    props.put("hello", "world");
    props.putIfAbsent("hello", "world");
    props.putAll(new HashMap<>());
    return props.get("Hello");
  }

After the quick-fix is applied:

  Object f(Properties props) {
    props.setProperty("hello", "world");
    props.putIfAbsent("hello", "world");
    props.putAll(new HashMap<>());
    return props.getProperty("hello");
  }
