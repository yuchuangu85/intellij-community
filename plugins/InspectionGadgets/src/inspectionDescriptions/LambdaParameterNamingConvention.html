Reports lambda parameters whose names are either too short, too long, or do not follow the specified regular expression pattern.

Example: Function<String, String> id = X -> X; should be reported if the inspection is enabled with default settings (where parameter name should start from lowercase letter)

Use the fields below to specify minimum length, maximum length and regular expression expected for lambda parameter names. Specify 0 to not check the length of names. Regular expressions are in standard java.util.regex format.
