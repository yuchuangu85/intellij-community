Reports any calls to a method with varargs where the call has a single argument in the vararg parameter position, which is either a null or an array of a subtype of the vararg parameter. Such an argument may be confusing as it is unclear if a varargs or non-varargs call is desired. For example:

Example:

  String[] ss = new String[]{"foo", "bar"};
  System.out.printf("%s", ss);

In this example only first element from array will be printed.
