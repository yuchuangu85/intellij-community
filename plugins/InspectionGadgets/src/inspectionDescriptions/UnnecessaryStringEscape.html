Reports unnecessarily escaped characters in String and optionally char literals. The escaped tab character \t is not reported, because it would otherwise be invisible.

Examples: 

  String s = "\'Scare\' quotes";
  String t = """
    All you need is\n\tLove\n""";

After the quick fix is applied the result looks like: 

  String s = "'Scare' quotes";
  String t = """
    All you need is
    \tLove
  """;

New in 2019.3
