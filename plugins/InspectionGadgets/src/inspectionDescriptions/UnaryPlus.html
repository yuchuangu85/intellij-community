Reports any uses of the unary + operator. Unary plus is usually a null operation, and its presence may represent a coding error particularly in combination with the increment operator +++, or the equals operator =+.

Example:

  void unaryPlus(int i) {
    int x = + +i;
  }

Two possible quick fixes are suggested here:

1) Remove + operators before i variable: 
  void unaryPlus(int i) {
    int x = i;
  }
2) Replace + operators with prefix increment operator: 
  void unaryPlus(int i) {
    int x = ++i;
  }

Use the checkbox below to only report when a unary plus is used together with a binary or another unary expression. This means the inspection won't warn when a unary plus expression is used for example as a method argument, or in an array initializer expression.
