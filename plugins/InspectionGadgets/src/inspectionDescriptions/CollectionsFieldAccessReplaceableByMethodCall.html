Reports usages of the java.util.Collections fields EMPTY_LIST, EMPTY_MAP or EMPTY_SET. These field usages can be replaced with method calls emptyList(), emptyMap() or emptySet(). Such method calls prevent "unchecked" warnings by the compiler because the type parameters can be inferred.

Example:

  List<Integer> emptyList = Collections.EMPTY_LIST;

After the quick-fix is applied, the result looks like this:

  List<Integer> emptyList = Collections.emptyList();

This inspection only reports if the language level of the project or module is 5 or higher
