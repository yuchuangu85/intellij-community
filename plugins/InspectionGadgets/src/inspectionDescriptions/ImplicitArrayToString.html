Reports any arrays used in String concatenations or as parameters to java.io.PrintStream methods (such as System.out.println()).

Usually in such cases, the content of the array is meant to be used and not the array object itself.

Example:

  void print(Object[] objects) {
    System.out.println(objects);
  }

After the quick-fix is applied the result looks like:

  void print(Object[] objects) {
    System.out.println(Arrays.toString(objects));
  }
