Reports on any expressions of type char which are used in addition or subtraction expressions. Such code is not necessarily a problem, but may result in bugs e.g., if usage as string is expected instead.

Example: int a = 'a' + 42;

After the quick-fix is applied, the result looks like this: int a = (int) 'a' + 42;

or for example in the String context: 

int i1 = 1;
int i2 = 2;
System.out.println(i2 + '-' + i1 + " = " + (i2 - i1));

After the quick-fix is applied, the result looks like this:

System.out.println(i2 + "-" + i1 + " = " + (i2 - i1));
