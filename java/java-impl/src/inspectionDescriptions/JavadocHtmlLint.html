Reports the same HTML issues in the Javadoc comments which are reported by the DocLint since Java 8.

The inspection detects issues like:

  • self-closed, unclosed, unknown, misplaced or empty tag
  • unknown or wrong attribute
  • misplaced text

Example:

/**
 * Unknown tag: List<String>
 * Unclosed tag: error</b>
 * Misplaced text or tag: <ul><li>one</li>, <li>two</li></ul>
 * Wrong or empty attribute: <a href=""></a>
 * Self-closed tag: <br/>
 * ...
 */
void sample(){ }
