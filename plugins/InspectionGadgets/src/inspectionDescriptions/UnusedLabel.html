Reports labels which are not the target of any break or continue statements.

For example: 


  label: for (int i = 0; i < 10; i++) {
    if (i == 3) {
      break;
    }
  }

After the quick fix is applied the result looks like: 


  for (int i = 0; i < 10; i++) {
    if (i == 3) {
      break;
    }
  }
