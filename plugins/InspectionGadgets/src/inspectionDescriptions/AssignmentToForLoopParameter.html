Reports assignment to, or modification of for statement parameters inside the body of a for loop. While occasionally intended, this construct can be confusing, and is often the result of a typo, or wrong variable being used.

Example:

  for (String s : list) {
    // Warning: s is changed inside the loop
    s = s.trim();
    System.out.println("String: " + s);
  }

A quick-fix is suggested to declare a new variable:

  for (String s : list) {
    String trimmed = s.trim();
    System.out.println("String: " + trimmed);
  }

Assignments in basic for loops without an update statement are not reported. In these cases the assignment is probably intended and can't be easily moved to the update part of the for loop.

Example:

  for (int i = 0; i < list.size(); ) {
    if (element.equals(list.get(i))) {
      list.remove(i);
    } else {
      // modification of for loop parameter is not reported
      // as there's no update statement
      i++;
    }
  }
