Reports any class initializers which may safely be made static. A class initializer may be static if it does not reference any of its class' non static methods and non static fields. Static initializers are executed once when the class is resolved, though non-static initializer are executed on each class instance creation.

Example:

  class A {
     public static String CONSTANT;
     {
        CONSTANT = "Hello";
     }
  }

After the quickfix is applied the result looks like:

  class A {
     public static String CONSTANT;
     static {
        CONSTANT = "Hello"; //initialized now only once per class
     }
  }
