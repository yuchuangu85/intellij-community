Reports any new expressions that instantiate utility classes. Utility classes have all fields and methods declared static. Instantiation of such classes is most likely unnecessary and indicates a mistake.

Example:

  class MyUtils {
    public static double cube(double x) {
      return x * x * x;
    }
  }
  class Main {
    public static void main(String[] args) {
      // Instantiation of utility class
      MyUtils utils = new MyUtils();
    }
  }

To avoid instantiations of utility classes completely, it's recommended to add a private default constructor.
