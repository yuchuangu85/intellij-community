Reports String.format call can be replaced with string templates.

Using a corresponding functions makes your code simpler.

The quick-fix replaces the call with string templates.

Example:


  fun main() {
      val id = "abc"
      val date = "123"
      val s = String.format("%s_%s_%s", id, date, id)
  }

After the quick-fix is applied:


  fun main() {
      val id = "abc"
      val date = "123"
      val s = "${id}_${date}_$id"
  }
