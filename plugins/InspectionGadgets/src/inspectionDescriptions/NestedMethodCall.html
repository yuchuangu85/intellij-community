Reports method calls used as parameters of another method call. The quick-fix introduces a variable to reduce cognitive complexity and make debugging easier.

Example:

  public int y() { return 1; }
  public int f(int x) { return 2 * x; }

  public void foo() {
      int x = f(y());
  }

After the quick-fix is applied, the result looks like this:

  public int y() { return 1; }
  public int f(int x) { return 2 * x; }

  public void foo() {
      int y = y();
      int x = f(y);
  }

Use the first checkbox below to ignore method calls in field initializers.

Use the second checkbox below to ignore calls to static methods.

Use the third checkbox below to ignore calls to simple getters.
