Reports malformed format strings in case when they don't fit the standard Java syntax.

By default, inspection consider a compile-time constant as a format string if it's used as argument to appropriate methods on java.util.Formatter, java.lang.String, java.io.PrintWriter or java.io.PrintStream.

Example:

  String.format("x = %d, y = %d", 42);

Use settings below to mark additional classes and methods as related to the string formatting.
