Reports comparison operations where the constant value is on the wrong side. Some coding conventions specify that constants should be on a specific side of a comparison, either left or right.

Example:

  boolean compare(int x) {
    return 1 > x; // Constant '1' on the left side of the comparison
  }

After the quick-fix is applied, the result looks like this:

  boolean compare(int x) {
    return x < 1;
  }

Use the combo box below to choose the side of constants in comparisons.
