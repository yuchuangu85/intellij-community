Reports redundant Stream or Optional calls like map(x -> x) or filter(x -> true), redundant sorted or distinct.

Note that a mapping operation in code like streamOfIntegers.map(Integer::valueOf) works as requireNonNull check: if the stream contains null, it will throw a NullPointerException, thus it's not absolutely redundant. Uncheck the checkbox below if you don't want such cases to be reported.

This inspection only reports if the language level of the project or module is 8 or higher
