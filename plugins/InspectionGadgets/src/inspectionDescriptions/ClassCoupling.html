Reports classes that are highly coupled, i.e. that reference too many other classes. Classes with too high coupling can be very fragile, and should probably be split to smaller classes.

Use the field below to specify the maximum coupling a class is allowed to have.

Use the first checkbox below to specify whether the references to system classes (those in the java.or javax. packages), should be counted.

Use the second checkbox below to specify whether the references to any library classes should be counted.
